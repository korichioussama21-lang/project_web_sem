🚀 Prochaines étapes

  1. Installer les dépendances: npm install
  2. Compiler les modules: npm run build
  3. Tester les démos:
    • npm run demo:rdf
    • npm run demo:ontology
    • npm run demo:sparql
    • npm run demo:reasoning
  4. Consulter la documentation dans docs/members/
